## 贰.  

    假如人生不曾相遇
    我还是那个我
    偶尔做做梦
    然后
    淹没在这喧嚣的城市里
    我不会了解
    这个世界还有这样的一个你
    让我回味
    让我心醉
    假如人生不曾相遇
    我不会相信
    有一种人可以百看不厌
    有一种人一认识就觉得温暖。
