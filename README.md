## 廿.  

    结婚以前大胡子问过我一句很奇怪的话：  
    “你要一个赚多少钱的丈夫？”  
    我说：“看得不顺眼的话，千万富翁也不嫁；看得中意，亿万富翁也嫁。”  
    “说来说去，你总想嫁有钱的。”  
    “也有例外的时候。”我叹了口气。  
    “如果跟我呢？”他很自然地问。  
    “那只要吃得饱的钱也算了。”  
    他思索了一下，又问：“你吃得多吗？”  
    我十分小心地回答：“不多，不多，以后还可以少吃点。”  
    就这几句对话，我就成了大胡子荷西的太太。
